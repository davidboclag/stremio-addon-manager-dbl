# üß© Stremio Addon Manager DBL

**Stremio Addon Manager DBL** is a web tool designed to simplify the management, configuration, and synchronization of addons in your **Stremio** account.  
With an intuitive interface, you can back up and restore your setup, install recommended addons, and fully customize your streaming experience.

---

## üöÄ Main Features

### üîß What can you do on the website?

- **Log in with your Stremio account**
- **Choose between different preconfiguration options**  (free or debrid-based service)
- **Select your preferred language** and apply the chosen setup
- **Automatically install recommended addons**, including *Cinemeta* and *anime addons* (optional)
- **Set subtitle language** using the *SubHero* addon
- **Restore Stremio to its original state**
- **Export your current configuration** to create a backup *(highly recommended the first time you use the site)*
- **Import a previously saved backup**
- **Update your list of installed addons** at any time
- **Remove addons individually or all at once**
- **Sync your configuration with Stremio** to reflect changes instantly
- **Load a complete preconfiguration** with all recommended addons  
   ideal for manual installation or tweaking default values ‚Äî the system automatically refreshes addons when changing language or debrid token)*
